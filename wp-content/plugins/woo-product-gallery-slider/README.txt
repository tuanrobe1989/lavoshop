== Installation ==


1. Upload `woo-product-gallery-slider` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

=How can i use it with page builders?=
the free version is not working with the custom page which made by elementor or other builders such as : Divi ,beaver builder & wpbakery page builder. but the pro version is works nicely with page builders. [Click Here to Get the Pro Version](http://codeixer.com/twist)
=Is the plugin compatible with my theme?=
we're using this 'woocommerce_before_single_product_summary' this hook to override the gallery.if your theme is using default WooCommerce hook for single-product.php page then this item will works perfectly.
